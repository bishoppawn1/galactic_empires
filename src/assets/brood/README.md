# Brood unit art

The transparent ground-unit sprites were generated with the built-in image generation tool on a removable magenta background, then processed with the imagegen skill's chroma-removal helper and normalized to 320 × 320 pixels.

Each organism uses a dedicated silhouette built around its battlefield role: clustered Broodlings, ranged acid sacs, a sensory-heavy Skitterer, armored beasts, living artillery, a crowned Synapse Guard, a dorsal Siege Crawler cannon, and a rooted Spine Tower. The shared olive chitin, ivory bone, and acidic-yellow bioluminescence preserve a coherent faction language without reusing Coalition troop art.
