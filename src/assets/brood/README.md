# Brood unit art

The transparent ground-unit and spaceship sprites were generated with the built-in image generation tool on a removable magenta background, then processed with the imagegen skill's chroma-removal helper and normalized to 320 × 320 pixels.

Each organism uses a dedicated silhouette built around its battlefield role: clustered Broodlings, ranged acid sacs, a sensory-heavy Skitterer, armored beasts, living artillery, a crowned Synapse Guard, a dorsal Siege Crawler cannon, and a rooted Spine Tower. The shared olive chitin, ivory bone, and acidic-yellow bioluminescence preserve a coherent faction language without reusing Coalition troop art.

The fleet follows the same language with role-specific top-down silhouettes: the podded Spore Ark, hooked Claw Frigate, needle-thin sniper, neural-crowned Hive Cruiser, manta-like Void Stalker Cruiser, incubation-heavy Brood Mega-Carrier, jawed Leviathan, and maw-fronted World Eater. Every hull points upward in its source sprite so the shared heading rotation continues to face ships along their movement path.
